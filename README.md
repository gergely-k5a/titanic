# Titanic

Visualize data of passengers aboard the legendary ship

## Installation

```
npm i
```

## Running

### Run the server and client in development mode

```
npm run dev
```

### Run the server

```
npm start
```

### Run the client

```
npm run client
```

### Run tests on client

```
cd client
npm run test
```

## Possible improvements
